# House Price Prediction

## Overview
A beginner-level Machine Learning project that predicts house prices using Linear Regression

## Dataset
Housing dataset with features like area, bedrooms, bathrooms , stories and amentitites

## Feature Used
-Area(square footage)
-Bedrooms
-Bathrooms
-Stories
-Air Conditioning
-Preferred Area
-Furninshing Status

##Results
-R SQUARE :0.65
-RMSE: 13.24 LAKHS
